# Prodigy_WD_2
# ⏱️ Stopwatch Web Application

A modern and responsive **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**. This project allows users to accurately measure time intervals with essential stopwatch features such as **Start**, **Pause**, **Reset**, and **Lap Time Tracking**.

## 🚀 Features

* ▶️ Start the stopwatch
* ⏸️ Pause and resume the timer
* 🔄 Reset the stopwatch
* 📝 Record multiple lap times
* 🎨 Modern and responsive user interface
* ✨ Smooth animations and interactive button effects
* 📱 Mobile-friendly design

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling, animations, and responsive layout
* **JavaScript (ES6)** – Stopwatch functionality and lap tracking
